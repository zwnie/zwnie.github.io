Hyperdynamics

发表于 2021-12-01 更新于 2023-12-19 分类于 dl-poly
本文字数： 3.5k 阅读时长 ≈ 3 分钟

在固体动力学（如扩散）过程中，构型可能长时间被困在势能面的一个谷，难以观察到原子跳跃。Hyperdynamics就是用来加速跳跃频率的手段。

Nudged Elastic Band (NEB)

NEB用来寻找反应路径，在这个过程中产生活化能。具体步骤如下：

Full Path Kinetics

Full Path Kinetics通过加速的手段寻找其它构型，计算相变的活化能和相变时间，随后用以确定系统的完整动力学。算法如下：

  1. 寻找一个局部极小构型A，能量$V{0}$，随后加一个偏置势进行模拟，收集过程中的boost factor $e^{\beta W{bias} (R^{N})}$
  2. 经历一段时间间隔（’BPD block’），优化此时的结构，得结构B，B与A相比，如果有原子的位移超过了用户预设的distance（’catch radius’），则认为相变发生。如果相变发生，可选择性的执行NEB，计算反应的活化能。
  3. 一旦相变发生，对时间区间内的所有结构进行优化，寻找最初发生相变的时间，从而反推实际相变时间。
  4. B作为初始构型，重新标定$V_{0}$，进行模拟（参考状态跟随扩散路径）
  5. 遍历所有状态，最终得出A的扩散路径

Configuration Sampling

优势在于能更快的遍历全空间。在构型A的基础上，加一个偏置势，$V_{0}$不会再改变，进行一段时间的模拟，最后利用boost factor计算力学量的期望值。

可以看到，Arrhenius law给出了高温和低温下化学反应速率常数的关系，进而给出了状态寿命的关系。由此可以考虑，对低温下难以发生的化学反应$A \rightarrow B$，可以在高温下较快发生，然后通过$\tau{High}$反推$\tau{Low}$，这就是TAD的基本思想。

  1. 优化结构得到状态A
  2. 高温下进行模拟，判断是否发生相变。如果是，NEB计算活化能
  3. 对相变前的所有结构，优化后判断相变发生时间$\tau_{High}$
  4. 利用下式反推$\tau_{Low}$
  5. 重新将体系拨回A态、重置速度，寻找其它态B, C, D…
  6. 在高温下需要一个截止时间$t{stop}$，认为在$t{stop}$内已经找到了低温下的最快反应路径
  7. 从A在低温下的最快反应路径B出发，重复上述流程，寻找B的最快反应路径，从而建立体系反应路径$A \rightarrow B \rightarrow C \rightarrow …$
